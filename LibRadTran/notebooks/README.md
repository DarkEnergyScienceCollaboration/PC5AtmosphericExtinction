PC5AtmosphericExtinction/LibRadTran/notebooks/README.md
-------------------------------------------------------

Directory containing notebooks to control input and output of LibRadtran.


These notebook are written for python 2.7
Packages required:

- numpy
- matplotlib
- astropy

-------------------
